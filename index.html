Mind Map Studio

Fully Offline • No Data Leaves This Device
Designed for privacy-sensitive settings This tool runs entirely in your browser with no internet connection required. No data is transmitted, stored remotely, or accessible to anyone else. It is suitable for environments where putting information into online tools would be inappropriate — including counselling, clinical, therapeutic, custodial, and other confidential settings.

How It Works

  1. 1Type phrases in the side panel (one per line) and tap Add to Canvas. Each phrase becomes a movable card.
  2. 2Drag cards anywhere to arrange them. Drag empty space to pan the canvas. Use the zoom controls (bottom-right), mouse wheel, or pinch to zoom. Double-tap a card to edit its text.
  3. 3Switch to Connect mode and tap two cards to draw a line between them. Tap a line to remove it.
  4. 4Switch to Label mode and tap the canvas to add free text annotations.
  5. 5Use Shuffle to randomly rearrange cards, or Float to set them drifting — tap a card to freeze it in place as ideas emerge.
  6. 6When you’re happy, tap Save as Image to download a PNG.
Tip: You can change card colours, line colours, line thickness, and label text size from the side panel. Use the ⊞ button (bottom-right) to fit all cards in view. On desktop, collapse the panel with the arrow button for more canvas space. On mobile, tap the menu icon (top-left) to open the panel.
